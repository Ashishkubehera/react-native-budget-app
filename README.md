# React Native budget app

![budgetapp](https://image.ibb.co/c1Rknx/budgety.png)

## Setup

1. Clone Repository
2. `cd react-native-budget-app`
3. `npm i`
4. `expo start`

## Includes

- react
- react-native
- redux
- react-redux
- react-native-elements
- react-navigation
- uuid
- redux-persist

## Features

- add month
- remove month
- add incomes/costs to month
- remove incomes/costs
- count percentage costs
